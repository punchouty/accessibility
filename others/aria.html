HTML4 and WAI-ARIA

It make Rich Internet applications more accessible to people with disabilities. It especially helps with dynamic content and advanced user interface controls developed with Ajax, HTML, JavaScript, and related technologies.

No Semantic Meaning

Using DIV tags : Airtel.

Using HTML tables Reliance Communication. Never user HTML tables for layout

Example markup

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
	<title>Demo Layout</title>
	<link href="css/some.css" rel="stylesheet">
    <script src="js/some.js" type="text/javascript"></script>
<head>
<body>
	<div class="header">
		<!-- some html code -->
	</div>
	<div class="navigation">
		<!-- some html code -->
	</div>
	<div class="main">
		<div class="side-navigation">			
			<!-- some html code -->
		</div>
		<div class="content">
			<!-- some html code -->
		</div>
	</div>
	<div class="footer">
		<!-- some html code -->
	</div>
	
</body>
</html>

Example Markup with ARIA

<!DOCTYPE html >
<html>
<head>
	<title>Demo Layout</title>
	<link href="css/some.css" rel="stylesheet">
    <script src="js/some.js" type="text/javascript"></script>
<head>
<body>
	<div role="banner" class="header">
		<!-- some html code -->
	</div>
	<div role="navigation" class="navigation">
		<!-- some html code -->
	</div>
	<div role="main" class="main">
		<div role="complementary" class="side-navigation">			
			<!-- some html code -->
		</div>
		<div role="region" class="content">
			<!-- some html code -->
		</div>
	</div>
	<div  role="contentinfo" class="footer">
		<!-- some html code -->
	</div>
	
</body>
</html>
