Guideline 1.1 Text Alternatives

Provide text alternatives for any non-text content so that it can be changed into other forms people need, such as large print, braille, speech, symbols or simpler language

Links - alt text in href tag

  • How to make it Accessible
  • Best Practices
  • No alt text at all - LIC India
  • Irrelevant alt text : Look at alt text under "CONTACT US" image - LIC India
  • Alt text for image when it is not required - spacer, horizontal and vertical line images etc.
  • Using CSS for image display : Look at footer for payment options - Snapdeal
Look at weather information at http://bbc.com or any other image 
<img alt="Sunny" title="Sunny" src="url" class="icon">
Spacer are used properly at LIC India 
<img src="pages/spacer.gif" width="36" height="1" alt="">
  1. Always provide the descriptive and relevant text in alt tag
  2. Give alt="" for - utility images like spacer, horizontal and vertical line
  3. Use CSS to display images for specialise icons and logo only (Look for repeating behaviour). Avoid using CSS to display general images
