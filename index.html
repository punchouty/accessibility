Accessibility Workshop

Specifically design for Web Designer and Developers, this two days hands on workshop cover WCAG 2.0 guidelines and standards, WAI-ARIA specifications and practices along with HTML5 accessibility features in depth.

WCAG 2.0

Explain Principles, Guidelines, Success Criteria, Sufficient and Advisory Techniques in easy to follow course with plenty of examples

WAI-ARIA

WAI-ARIA, the Accessible Rich Internet Applications Suite, helps with dynamic content and advanced user interface controls developed with Ajax, HTML, JavaScript, and related technologies. We teach developer to how to comply with these standards.
